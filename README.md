# 🚀 Sistema de cine

Este proyecto consiste en el desarrollo de un sistema de gestión integral para cines, diseñado para realizar
operaciones fundamentales mediante una arquitectura de microservicios. El sistema aborda la necesidad de
agrupar procesos, como el control de inventario, películas, comentarios, salas, la programación de funciones y la
gestión de accesos según el perfil del usuario. Además, el sistema garantiza la integridad de los datos de las
ventas de las entradas y confiterías, permitiendo una administración eficiente de los ingresos. Este sistema
beneficiaria principalmente al personal de administración.

## 🛠️ Microservicios

* Autenticacion
* Usuario
* Pelicula
* Funciones
* Entradas
* Comentario
* Confiteria
* Inventario
* Objetos perdidos
* Sala
